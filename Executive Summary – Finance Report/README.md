# Executive Summary Finance - Power BI Project

This Power BI project provides an executive summary of financial performance using the **Financial Sample** dataset. It includes key metrics such as profit, sales, and segment performance, visualized across multiple dashboards.

## Dataset

The dataset contains financial data with the following dimensions and metrics:

- **Metrics:** Profit, Total Sales
- **Dimensions:** Month, Year, Country, Product, Segment
- **Format:** Excel (.xlsx)
- **Filename:** `Financial_Sample.xlsx`

## Power BI Visualizations

The project includes the following dashboards and charts:

1. **Profit by Month and Year**
   - Displays monthly and yearly profits.
   - Helps identify seasonal and annual trends in profitability.

2. **Total Profit by Country**
   - Shows total profit generated in each country.
   - Useful for regional financial performance analysis.

3. **Total Sales by Product and Segment**
   - Breaks down total sales by product and business segment.
   - Highlights top-performing products and segments.

## Key Insights

- Provides a clear overview of profit trends by month and year.
- Identifies high-performing countries and products.
- Supports executive-level decision-making by summarizing financial performance across multiple dimensions.

## Usage

- Open the provided Excel file in Power BI to explore or recreate dashboards.
- Use interactive filters to analyze profit and sales by time, country, product, and segment.
- Ideal for financial analysis, executive reporting, and data visualization practice.

## File Information

- **Dataset Filename:** `Financial_Sample.xlsx`
- **Format:** Excel (.xlsx)
- **Contents:** Financial metrics including profit and sales by month, year, country, product, and segment.
